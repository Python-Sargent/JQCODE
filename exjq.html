Introduction JQuery:

Jquery is a JavaScript library for easily scripting webpages.

JQuery can be used to quickly do many different things, linke hide elements:

$("p").hide();

If at any time you would like to try the code on this site you can use GDB Debugger or W3Schools Tryit Editor or use a text editor. To do that make a new file called 'index.html' and open with your text editor (don't double click, it will open the HTML file in your browser)

JQuery Syntax Basics

Jqury is a fun language to learn but you need to remember a few important things:

If you don't know JavaScript i'd advise learning that first.
Jquery uses:

$("selector").action();

to manipulate elements, notice the $.

lets make a script:

$("p").hide();

This script will hide all the paragraphs, notice the "p".

Now lets put it to action by making it wait for the paragraph to be clicked.
(the first part is to make it only work when the page is loaded)

$(document).ready(function(){
  $("p").click(function(){
    $(this).hide();
  });
});
