CSS Syntax Basics

CSS is a easy language to learn but you need to remember a few important things:

To start using CSS you first need to know how it works.
CSS uses a system of:

selector {
  attribute: value;
}

Lets turn that into real code

p{color: red;}

Lets change some more attributes:

p {
  color: red;
  background-color: blue;
  padding: 5px;
  border-radius: 15px;
  font-size: 20px;
}

To see it in action use one the editors above or go Here

You can also use 'Inline' CSS which looks like this:

<p style="color: red;">This paragraph will be red<p>

output: 
This paragraph will be red
