Introduction HTML:

HTML uses tags. tags start with a '<' and end with a '>'.
Most tags have a starting tag: <tag_name>,
and an ending tag: </tag_name> notice the slash.

<p>Hello World</p>

More on tags:

TAG TYPE SYNTAX
<p> Paragraph Start-End
<a> Link Start-End
<i> Italicised Start-End
<b> Bold Start-End
<div> Divided Area Start-End
<img> Image Start
<hr> Horizontal Rule Start
<button> Button Start-End

HTML Syntax Basics

HTML is a easy language to learn but you need to remember a few important things:

To start making your first page we need to tell the computer what kind of file this is (.html):

<!DOCTYPE html>

Then we have to put in some important tags <html>, <head>, and <body>

<!DOCTYPE html>
<html>
  <head>
    <title>CodeExample.com</title>
  </head>
  <body>
    <p>This is a paragraph</p>
  </body>
</html>
